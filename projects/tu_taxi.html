Tu taxi Morelia

"Tu taxi Morelia", es una aplicación móvil que permite al usuario solicitar un servicio de taxi para trasladarse entre dos puntos de su ciudad.

aptitudes:
Flutter
Firebase
Javascript
Node js
Mobile developing
Web developing

En su pantalla principal, el conductor tiene un mapa donde puede ver su ubicación actual, así como un botón con el que puede terminar su turno de trabajo.

El conductor recibe notificaciones de viaje en tiempo real, incluso si se encuentra fuera de la aplicación.

Antes de solicitar un viaje, el cliente puede visualizar la ruta hasta su destino, así como el costo del servicio.

Una vez iniciado el viaje, el usuario puede visualizar la ruta que el conductor está siguiendo.

El usuario puede comunicarse vía chat con su conductor durante todo el servicio.

Tanto el cliente como el conductor disponen de un 'botón de pánico', a través del cual pueden informar de la ocurrencia de un accidente o situación de peligro.

Una vez finalizado el viaje, el usuario puede evaluar a su conductor y el servicio que este le brindó
