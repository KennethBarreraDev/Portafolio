Dolphin

Aplicación móvil de aprendizaje de idiomas que emplea tecnologías de reconocmiento de voz, inteligencia artificial y machine learning para ayudar al usuario a mejorar su pronunciación y gramática.

aptitudes:
Flutter
Firebase
Javascript
Node js
Mobile developing
Web developing

El usuario puede seleccionar el idioma que desea aprender o ingresar un código de grupo para inscribirse a él.

Dolphin tiene un juego de memoria que consiste en arrastrar y soltar las letras mostradas para formar la palabra alusiva a la imagen.

Para un aprendizaje efectivo, la aplicación posee lecciones de gramática, escritura, listening y speaking.

Las lecciones de Listening tienen preguntas que ayudan a evaluar la comprensión del alumno. Las voces de los audio fueron generadas mediante IA.

A través de las lecciones de Speaking, se detecta la pronunciación del usuario y se indica en que medida se parece lo dicho a la respuesta correcta.
