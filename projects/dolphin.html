Dolphin

Mobile language learning application that uses AI technologies to detect and improve the user's pronunciation

aptitudes:
Flutter
Firebase
Javascript
Node js
Mobile developing
Web developing

The user can select the language him/her want to learn as well as enter a code of a class group to join it

Dolphin has a memory game that consists of dragging and dropping the letters shown to form the word that alludes to the image

For effective learning, the application has grammar, listening, writing and speaking lessons

Listening lessons have questions that help assess the user's understanding. Audio voices were generated by AI

Through the speaking lessons, the user's pronunciation is detected and it is indicated how much what was said is similar to the correct answer.
